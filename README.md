# Bank Management Android App

The Bank Management Android App is an Android application that allows users to manage their bank accounts and interact with various banking services. The app provides features such as user login, transaction management, and a map integration using OpenStreetMap to locate different bank branches. Users can also contact the bank's call center, send SMS messages, or send emails for assistance.

## Features

The Bank Management Android App offers the following features:

1. **User Authentication**: Users can create an account or log in using their credentials to securely access their bank accounts.

2. **Transaction Management**: Users can view their transaction history, including details such as transaction amount, date, and description. They can also filter transactions based on various criteria and perform actions such as adding a new transaction or editing existing ones.

3. **Branch Locator**: The app integrates OpenStreetMap to display a map that shows the locations of different bank branches. Users can easily find the nearest branch based on their current location or search for a specific branch.

4. **Contact Options**: Users can conveniently contact the bank's call center directly from within the app. They can make a phone call, send an SMS, or compose an email to reach out for assistance or inquiries.

## Getting Started

To use the Bank Management Android App, follow these steps:

1. Clone the repository to your local machine.

```
git clone https://github.com/your-username/bank-management-android.git
```

2. Open the project in Android Studio.

3. Build and run the app on an Android emulator or physical device.

## Technologies Used

The Bank Management Android App is built using the following technologies and frameworks:

- Android Studio: The primary development environment for building Android applications.

- OpenStreetMap: An open-source map platform used for displaying and interacting with map data.
